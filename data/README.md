# Data

This project uses Centers for Medicare & Medicaid Services 2026 Health Insurance Exchange Public Use Files.

Raw zip files are downloaded to `data/raw/` during a run and removed by default after outputs are generated. Re-run `python3 run_marketplace_plan_analysis.py` to download fresh source files and regenerate the report.

Primary source page:

https://www.cms.gov/marketplace/resources/data/public-use-files
